ElGamal Cryptosystem

ElGamal is probabilistic (same message → different ciphertext every time) and multiplicatively malleable — a known weakness you can exploit below.

1

Non-Determinism Demo

Encrypts the same message twice. Both results differ because of a random ephemeral key k chosen per encryption.

2

Malleability Attack

Given ciphertext (C1, C2) = E(m), an attacker can compute E(k·m) for any k without knowing the private key or m. This is the multiplicative malleability weakness.

ECC Playground

Elliptic curves provide equivalent security to RSA with much shorter keys. The core operation is scalar multiplication Q = k·P — easy to compute, practically impossible to reverse.
